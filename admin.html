GitHub token (one-time setup)

Publishing commits directly to DmitryGofman/DGLabs needs a GitHub token stored in this browser only:

  1. Open github.com → Settings → Fine-grained tokens → Generate
  2. Repository access: Only select repositories → DGLabs
  3. Permissions → Contents: Read and write. Generate & copy.
